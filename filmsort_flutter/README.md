# FilmSort Flutter App

This folder contains the Dart/Flutter code intended to build FilmSort into a mobile application for Android and iOS. 

## Features Integrated (Skeleton):
- Material Design & Custom AppColors (`#050505` background, `#ff6b00` brand orange)
- Bottom Navigation Routing (Home, Search, Library, Favorites)
- Hero section matching web layout
- Horizontal scrolling library catalog rows

## Running this app

1. Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your local machine.
2. In Google AI Studio, open **Export > Export to ZIP**.
3. Extract the contents on your local machine and navigate into `filmsort_flutter/`.
4. Run `flutter pub get` to download dependencies.
5. Run the app in your connected device or emulator using `flutter run`.
